I've had lots of fun using `opencode` and `claude --dangerously-skip-permissions` to do my job.

I realised that customising the agent harness -the tools surrounding the llm- proves just as valuable as using a better model.

I use the tools to do all the work I couldn't care less about but I want to be an expert on the bits that are important.

so I'm building a [learning agent](github.com/luqs1/learning-agent).

## iteration 1: a system prompt for opencode

build agent comes with some junk in the system prompt and a skill would never have been enough. so I [defined a new](https://opencode.ai/docs/agents/#markdown) one. it makes me blush to think that some text is even worth talking about but the results are what matter.
